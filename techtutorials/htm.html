How to create a html file


Follow these steps to create a html file:

1.On your desktop right click anywhere.

2.Then select 'New Text Document' from the new option.

3.Then a notepad file will open.

4.On top left click on save as option.

5.Then save the file as .html or .htm extension in desirable place.

6.Now write the html code in your opened notepad file.

7.After saving it to preview that file go to that file that you saved with .html or .htm extension

8.Now open that file and see the preview in your browser.
